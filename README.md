# Infotact_Project1

AI-Powered Citizen Grievance Analysis System

📌 Project Overview

This project focuses on analyzing citizen complaints using Natural Language Processing (NLP) and Machine Learning techniques. The system classifies complaints into categories and helps identify patterns in public issues.

🎯 Objectives

- To preprocess and clean textual complaint data
- To perform Exploratory Data Analysis (EDA)
- To extract meaningful insights from complaint data
- To build machine learning models for classification
- To analyze complaint patterns and trends

🛠️ Technologies Used

- Python
- Pandas, NumPy
- NLTK (NLP)
- Scikit-learn (Machine Learning)
- Matplotlib, Seaborn (Visualization)
- Streamlit (UI)

📊 Dataset

The dataset used is a citizen complaint dataset (311 Service Requests) containing various complaint types.

⚙️ Methodology

🔹 Week 1: Text Preprocessing

- Converted text to lowercase
- Removed special characters
- Tokenization
- Stopwords removal
- Lemmatization

🔹 Week 2: Exploratory Data Analysis

- Category distribution analysis
- Complaint length analysis
- WordCloud visualization
- Identification of key patterns

🔹 Week 3: Feature Engineering & Modeling

- TF-IDF Vectorization
- Model training:
  - Logistic Regression
  - Naive Bayes
  - Decision Tree
  - Random Forest

🔹 Week 4: Evaluation & Deployment

- Model evaluation (Accuracy, Classification Report)
- Confusion Matrix
- Streamlit UI for prediction
